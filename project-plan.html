Overview

A portfolio website showcasing tutorials, projects, and coding solutions with an admin panel for content management. This document tracks the entire build process from planning to deployment.

Aspect Details
Purpose Developer portfolio + Tutorial hub + Project showcase
Target Audience Potential employers, fellow developers, coding learners
Key Features Interactive tutorials, LeetCode solutions browser, Arc Raiders dashboard, Admin panel
Design Inspiration akis.studio - Bold typography, smooth animations, dark theme

Architecture

The application follows a modern full-stack architecture with clear separation between the API backend and React frontend.

Tech Stack

Layer Technology Purpose
Frontend React 18 + TypeScript + Vite Fast, modern UI with type safety
Animations Framer Motion + GSAP Smooth, professional animations
Styling Tailwind CSS Utility-first, rapid development
Backend ASP.NET Core 8 Web API Robust, performant API
Database SQL Server (Azure SQL) Reliable data storage
Auth ASP.NET Core Identity + JWT Secure authentication
File Storage Azure Blob Storage PDFs, images, tutorial files
Hosting Azure App Service Scalable cloud hosting

Phases

The project is broken into four main phases, each with smaller manageable chunks.

Phase 1: Landing Page + About Me + Docker Tutorial â±ï¸ 3-4 days

Goal: Get something live and impressive ASAP

  • Project scaffolding (API + React)
  • Database schema design & migrations
  • Landing page with animations (akis.studio style)
  • Hero section with name/title
  • Projects showcase grid
  • About Me section with skills display
  • Social links (GitHub, LinkedIn, Coming Soon for others)
  • Resume display + PDF download
  • Docker tutorial integration
  • Deploy to Azure (staging)

Chunk Breakdown

1A Project scaffolding - empty ASP.NET Core API + React app 30 min
1B Database setup - DbContext, entities, migrations 1 hour
1C Landing page - Hero section only 1-2 hours
1D Landing page - Projects grid section 1-2 hours
1E Landing page - About preview + Skills 1-2 hours
1F Landing page - Footer with social links 30 min
1G API endpoints - GET projects, profile, tutorials 1 hour
1H Connect React to API - Replace hardcoded data 1 hour
1I About Me page - Full page with resume 1-2 hours
1J Docker tutorial integration 30 min
1K Deploy to Azure (staging) 1 hour
Phase 2: Admin Panel + Auth â±ï¸ 2-3 days

Goal: Manage content without touching code

  • User authentication (register/login)
  • Admin dashboard interface
  • Tutorial management (add/edit/delete)
  • About Me editor
  • Resume PDF upload
  • Project management
  • Role-based authorization (Admin vs Public)
  • Azure Blob Storage integration
  • Protected API endpoints
Phase 3: Arc Raiders Dashboard â±ï¸ 2-3 days

Goal: Showcase the game analytics project

  • Migrate Arc Raiders database to Azure SQL
  • API endpoints for Arc Raiders data
  • Interactive dashboard page
  • Items browser with filtering
  • Quest tracker
  • Event timers
  • Interactive maps
  • Project showcase page
Phase 4: LeetCode Solutions â±ï¸ 2 days

Goal: Searchable collection of coding solutions

  • Database tables for problems/solutions
  • Bulk import of existing 25 solutions
  • Solutions browser page
  • Filter by difficulty (Easy/Medium/Hard)
  • Filter by topic (Arrays, Trees, DP, etc.)
  • Search by name/number
  • Solution detail page with syntax highlighting
  • Explanation + complexity analysis
  • Admin: Add new solutions

Design Notes

Landing Page Sections

Hero - Full viewport, bold typography, subtle particle/gradient animation
Featured Projects - Horizontal scroll or staggered grid with hover effects
About Preview - Split layout with image/text, scroll-triggered animations
Skills - Animated bars or creative visualization
Contact/Social - Minimal footer with links

Animation Techniques

  • Scroll-triggered reveals (Framer Motion + Intersection Observer)
  • Smooth page transitions
  • Magnetic buttons (cursor follows)
  • Text split animations (letters animate in)
  • Parallax scrolling on images
  • Custom cursor
  • Loading screen with logo animation
